# 📚 子供の学習アプリ

子供向けの学習計画・管理Webアプリケーションです。モバイルでも使いやすいレスポンシブデザインで、楽しく学習を管理できます。

## 🌟 主な機能

- **学習タスクの管理**: 学習内容、教科、難易度を設定してタスクを追加
- **進捗状況の可視化**: ダッシュボードで全体の進捗をひと目で確認
- **タスクフィルター**: すべて/未完了/完了でタスクを絞り込み
- **データ永続化**: LocalStorageでデータを保存（ブラウザを閉じても消えません）
- **レスポンシブデザイン**: スマートフォン、タブレット、PCで快適に使用可能
- **子供向けUI**: カラフルで分かりやすいインターフェース

## 💡 使用方法

1. **タスクの追加**
   - 学習内容を入力
   - 教科を選択
   - 難易度（かんたん/ふつう/むずかしい）を選択
   - 「追加する」ボタンをクリック

2. **タスクの管理**
   - チェックボックスをクリックして完了/未完了を切り替え
   - ゴミ箱アイコンをクリックしてタスクを削除

3. **進捗の確認**
   - ダッシュボードで全タスク数、未完了、完了数を確認
   - 進捗バーで達成率を確認
   - 応援メッセージで励ましを受け取る

## 🎨 デザインの特徴

- **カラフルなグラデーション**: 紫系のグラデーションで楽しい雰囲気
- **大きなアイコン**: 子供が見やすい絵文字とアイコン
- **分かりやすいボタン**: タップしやすい大きさとデザイン
- **応援メッセージ**: 進捗に応じた励ましのメッセージ

## 📱 モバイル対応

- 画面幅768px以下: タブレット向けレイアウト
- 画面幅480px以下: スマートフォン向けレイアウト
- タッチ操作に最適化されたUI

## 🔒 データの保存

- ブラウザのLocalStorageを使用
- データはブラウザ内に保存され、外部には送信されません
- ブラウザのキャッシュをクリアすると削除されます
